# Quick Start - Тестирование

## Запуск расширения (СЕЙЧАС)

### Самый простой способ:
1. **Нажмите F5** в текущем окне VS Code
2. Откроется новое окно "[Extension Development Host]"
3. В левой панели (Activity Bar) появится иконка **сетки** (4 квадрата)
4. Кликните на эту иконку — откроется панель "MY PROJECTS"

### Добавление первого проекта:
1. Нажмите **+** в заголовке панели
2. Введите название: `Мой первый проект`
3. Описание: `Тестовый проект для проверки`
4. Теги: `test, demo, frontend`
5. Выберите цвет: 🔵 Синий
6. Проект появится в виде карточки!

### Что проверить:
- ✅ Добавление проекта
- ✅ Отображение карточки с цветной границей слева
- ✅ Отображение тегов
- ✅ Редактирование (иконка ✏️ при наведении)
- ✅ Удаление (иконка 🗑️ при наведении)
- ✅ Переключение вида Карточки ↔ Таблица (иконка списка в заголовке)

---

## Альтернативный способ (если F5 не работает):

### macOS:
```bash
# Добавить VS Code в PATH (одноразово)
export PATH="/Applications/Visual Studio Code.app/Contents/Resources/app/bin:$PATH"

# Запустить Extension Development Host
code --extensionDevelopmentPath="$PWD" --new-window
```

### Или через меню VS Code:
1. `Run > Start Debugging` (или ⌘+Shift+D → F5)
2. Выберите "Run Extension"

## Скриншот того, что должно получиться:

```
┌─────────────────────────────────────────┐
│ MY PROJECTS              [+] [≡]       │
├─────────────────────────────────────────┤
│ ┃ 🔵 Мой первый проект      ✏️ 🗑️     │
│ ┃                                       │
│ ┃ Тестовый проект для проверки          │
│ ┃                                       │
│ ┃ [test] [demo] [frontend]              │
└─────────────────────────────────────────┘
```

**Синяя линия слева** = цвет проекта  
**Синяя точка** = дополнительный индикатор
